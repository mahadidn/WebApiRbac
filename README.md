# RBAC System – ASP.NET Core Web API

A **Role-Based Access Control (RBAC)** system built with **ASP.NET Core Web API (.NET 10)** following **Clean Architecture principles**.  
This project provides authentication using **JWT**, role and permission management, and is designed to run with **PostgreSQL** and **Docker**.

---

## Features

- User authentication using **JWT Token**
- Role management
- Assign roles to users
- Assign permissions to roles
- Authorization based on **roles and permissions**
- Clean Architecture implementation
- PostgreSQL database
- Docker containerization
- RESTful API design

---

## Tech Stack

- **.NET 10**
- **ASP.NET Core Web API**
- **PostgreSQL**
- **Entity Framework Core**
- **JWT Authentication**
- **Docker & Docker Compose**
- **Clean Architecture**
